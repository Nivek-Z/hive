## 设计概念

- **世界观**：无限瓷白世界，雾色融入天际。每个蜂巢 = 一簇白瓷六棱柱
  （中央纪念碑 = 蜂巢本体，环形螺旋排布的矮柱 = 文字频道），沿对角「林荫道」排开，
  点阵虚线连接各群落。私域（好友私信）= 林荫道反向的**圆柱**群落——方与圆区分公共/私密。
- **运镜**：登录后高空长镜头降落 → 切蜂巢时抬升-滑翔-降落的弧线飞行 →
  进频道时推近构图（目标点右移给面板让位）→ 驻留时呼吸漂移 + 鼠标视差。
- **浮层**：编辑部式排版。左上编号索引（00 私域 / 01.. 蜂巢），左下舞台铭牌
  （编号—名称—成员/频道/在线），右侧白瓷玻璃聊天面板，3D 锚定的频道标签悬浮在柱体上方。
- **唯一彩色**：琥珀 `#eda400`。在线萤点（蜜蜂）绕群落盘旋；消息抵达时群落脚下
  点阵池脉冲发光；活跃频道柱抬升并戴琥珀六角光环；正在输入的频道柱轻轻浮动。
- **字体**：Schibsted Grotesk（标题/正文拉丁）+ Spline Sans Mono（编号/微标签），
  woff2 已本地化到 `vendor/fonts/`，离线可用；中文回退系统字体栈。

## 已实现（第一轮 · 整体骨架）

- 登录 / 注册（`/auth/*`），token 复用经典版的 localStorage key（两套皮肤免重复登录）
- 蜂巢索引 + 3D 群落生成（升起动画）+ 点击柱体或标签切换/进入
- 频道历史（上滑加载更多）、发送（乐观渲染 + nonce 对账）、斜杠命令透传
- 实时：MSG_NEW / MSG_DELETED / REACTION_UPDATE / TYPING / PRESENCE / READY /
  FRIEND_EVENT / ACHIEVEMENT_UNLOCKED / EGG / HIVE_EVENT(简化为整巢刷新) / ERROR
- 回复（引用条）、撤回、表情回应点按切换、图片消息展示 + 灯箱
- 私域：私信会话列表 / 好友列表 / 接受好友申请 / 发起私聊（圆柱群落 + 标签）
- 未读角标（频道标签 / 蜂巢索引 / 私域）、已读上报、成就横幅、彩蛋全林荫道发光
- prefers-reduced-motion 降级（跳切代替飞行）、WebGL 不可用时回退提示
