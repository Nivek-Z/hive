# Hive 蜂巢 - Java 程序设计大作业进度快照

本分支是从最终项目状态回溯整理出的透明进度快照，用于展示三周内从接口设计到最终交付的演进过程。  
这些提交不是原始开发历史，提交信息和文档均标注为 reconstructed checkpoint。

## 当前快照：起步阶段

起步阶段只包含项目骨架和抽象接口定义，还没有控制器、业务实现、数据库访问实现或前端页面。

四位成员的模块边界：

- 张致硕：蜂巢管理、邀请、文件上传。
- 张凯文：频道、消息、搜索、WebSocket 实时通信。
- 虞沛远：角色权限、好友、私聊。
- 江民智：认证、用户、成就、Web 配置。

后续快照：

- `progress/start`：起步阶段，接口与模块边界。
- `progress/early`：初期阶段，核心账号、用户、蜂巢、邀请、文件模块成型。
- `progress/middle`：中期阶段，主要后端模块整合，加入频道、消息、权限、好友和私聊。
- `progress/late-final`：晚期阶段，恢复到最终项目状态并补齐测试、文档、静态页面和演示资源。

详细说明见 `docs/progress-snapshots.md`。
