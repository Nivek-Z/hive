# Hive TUI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 用 Go 为现有 Hive 实时聊天系统新增一个简约、键盘优先的终端聊天前端。

**Architecture:** 在仓库根目录新增独立 Go module `tui/`，通过 REST API 完成登录、蜂巢/频道/历史消息读取，通过 WebSocket 完成实时收发。代码按配置、模型、REST 客户端、WebSocket 帧、频道树、应用状态和 TUI 入口拆分，核心逻辑用 Go 单元测试覆盖。

**Tech Stack:** Go 1.25、Bubble Tea、Lip Gloss、Gorilla WebSocket、go-toml/v2、标准库 `net/http`/`httptest`。
